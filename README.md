# Technical Assessment

This is a small frontend-only Next.js application that implements an audit/compliance dashboard for TAHWUL.  
It includes a main dashboard, a document tracking view, and a details page, all driven by mock data and built with modern React and Tailwind-style utility classes.

---

## How to Run the Project

### 1. Prerequisites

- **Node.js**: v18+ recommended
- **npm**: comes with Node

### 2. Install dependencies

From the project root:

```bash
npm install
